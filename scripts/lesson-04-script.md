## Guion base

Ahora que ya sabes crear títulos y párrafos, vamos a agregar más elementos a nuestra página.

Primero veremos las listas.

Una lista sirve para mostrar varios elementos relacionados, por ejemplo hobbies, comidas favoritas o pasos para hacer una actividad.

En HTML podemos usar listas no ordenadas y listas ordenadas.

Una lista no ordenada usa la etiqueta ul. Cada elemento dentro de la lista usa la etiqueta li.

También aprenderemos a colocar imágenes con la etiqueta img.

La etiqueta img usa el atributo src, que indica la dirección de la imagen, y el atributo alt, que describe la imagen.

Finalmente, veremos los enlaces. Un enlace se crea con la etiqueta a y usa el atributo href.

## Pausa de práctica

Agrega a tu página una imagen, una lista de tres hobbies y un enlace a una página que te guste.
