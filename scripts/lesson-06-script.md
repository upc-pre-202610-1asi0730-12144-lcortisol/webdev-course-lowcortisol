## Guion base

En esta lección vamos a mejorar el diseño de nuestra página usando estilos simples.

Crearemos una tarjeta de perfil. Una tarjeta es un bloque visual que agrupa información, como un título, una descripción o una imagen.

Primero necesitamos colocar nuestro contenido dentro de un contenedor. Para eso usamos la etiqueta div.

También usaremos una clase llamada card.

Una clase sirve para identificar un elemento y luego aplicarle estilos desde CSS.

En CSS, cuando llamamos a una clase, usamos un punto antes del nombre.

La propiedad width define el ancho de la tarjeta.

La propiedad margin ayuda a separar la tarjeta y centrarla.

La propiedad padding agrega espacio dentro de la tarjeta.

La propiedad border agrega un borde.

La propiedad border-radius redondea las esquinas.

La propiedad text-align centra el texto.

Y background-color cambia el fondo de la tarjeta.

## Pausa de práctica

Modifica tu tarjeta cambiando el ancho, color de fondo, borde y redondeado de esquinas.
