## Guion base

En esta lección aprenderás a organizar texto dentro de una página web.

HTML tiene diferentes niveles de títulos. El título más importante es h1. Luego existen subtítulos como h2, h3, hasta h6.

No se usan solo para cambiar el tamaño de la letra. También sirven para organizar la información.

El h1 puede ser el título principal de la página. Los h2 pueden ser secciones importantes, como Sobre mí o Mis intereses.

También usaremos la etiqueta p, que sirve para escribir párrafos.

Vamos a crear una pequeña página de perfil.

## Consejo

Usa un solo h1 como título principal. Usa h2 para dividir secciones.

## Pausa de práctica

Crea una página con un título principal, una sección Sobre mí y una sección Mis intereses.
