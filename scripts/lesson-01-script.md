## Guion base

En esta lección aprenderás qué es una página web y cómo se construye por dentro.

Una página web es un documento que podemos ver desde un navegador como Chrome, Firefox, Safari o Edge.

Para crear una página web básica usamos dos tecnologías principales: HTML y CSS.

HTML sirve para crear la estructura y el contenido.

CSS sirve para mejorar el diseño visual.

Una forma sencilla de entenderlo es imaginar una casa. HTML sería la estructura de la casa. CSS sería la decoración.

Ahora veremos un ejemplo simple en CodePen. CodePen es una herramienta gratuita en línea, por eso no necesitamos instalar ningún programa.
