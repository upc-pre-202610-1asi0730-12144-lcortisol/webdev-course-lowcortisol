## Guion base

En esta última lección revisaremos algunos errores comunes y recomendaciones para seguir mejorando.

Cuando estamos empezando con HTML y CSS, es normal cometer errores. Lo importante es aprender a identificarlos y corregirlos.

Primer error común: olvidar cerrar etiquetas.

Segundo error común: escribir mal una etiqueta.

Tercer error común: escribir atributos sin comillas.

Cuarto error común: olvidar el punto cuando usamos una clase en CSS.

Quinto error común: usar demasiados colores, tamaños o fuentes al mismo tiempo.

Una página sencilla, clara y ordenada suele verse mejor que una página con demasiados estilos diferentes.

## Recomendaciones finales

1. Practica con cambios pequeños.
2. Revisa el código con calma.
3. Usa nombres simples.
4. Prueba tu página varias veces.
5. Guarda tus ejemplos para seguir practicando.

## Pausa final

Revisa tu proyecto y corrige una etiqueta, un color, un texto y un enlace.

## Cierre del curso

Con esto terminamos el curso “Crea tu primera página web con HTML y CSS”.
