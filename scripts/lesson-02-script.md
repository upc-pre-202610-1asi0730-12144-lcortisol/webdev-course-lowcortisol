## Guion base

En esta lección aprenderás cómo se escribe HTML.

HTML funciona usando etiquetas. Una etiqueta normalmente se escribe entre los símbolos menor que y mayor que.

Un elemento HTML normalmente tiene tres partes: una etiqueta de apertura, un contenido y una etiqueta de cierre.

También existen los atributos. Un atributo da información adicional a una etiqueta.

Por ejemplo, en un enlace, el atributo href indica a qué página queremos ir.

Ahora vamos a escribir un ejemplo en CodePen.
