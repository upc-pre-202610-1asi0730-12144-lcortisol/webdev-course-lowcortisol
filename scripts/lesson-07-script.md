## Guion base

En esta lección construiremos el proyecto final del curso: una página de perfil personal.

Usaremos todo lo aprendido en las lecciones anteriores: títulos, párrafos, imágenes, listas, enlaces y estilos CSS.

La página tendrá una tarjeta con una imagen, un título, una descripción, una lista de hobbies y un enlace.

Empezaremos con el HTML.

Primero creamos un contenedor principal.

Dentro colocamos una imagen.

Luego agregamos un título.

Después escribimos una descripción.

Más abajo agregamos una lista de hobbies.

Finalmente agregamos un enlace.

Luego aplicamos CSS para cambiar el fondo, centrar la tarjeta, redondear la imagen, ajustar colores y mejorar el diseño visual.

## Pausa de práctica final

Personaliza tu página cambiando el título, la descripción, los hobbies, los colores y el enlace.
