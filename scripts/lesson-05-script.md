## Guion base

En esta lección aprenderás qué es CSS y cómo se usa para mejorar el diseño de una página web.

Hasta ahora hemos trabajado con HTML. HTML nos ayuda a colocar contenido.

Pero si queremos cambiar colores, fondos, tamaños o tipos de letra, usamos CSS.

CSS funciona con tres partes básicas: selector, propiedad y valor.

El selector indica qué elemento queremos modificar.

La propiedad indica qué aspecto queremos cambiar.

El valor indica el cambio específico.

Por ejemplo, podemos decir que todos los títulos tengan color azul.

## Pausa de práctica

Cambia el color del título, el color del párrafo y el color de fondo de la página.
