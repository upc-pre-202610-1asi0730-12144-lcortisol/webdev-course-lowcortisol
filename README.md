# Crea tu primera página web con HTML y CSS

Curso introductorio de desarrollo web para estudiantes de secundaria.

## Producto

## Team Leader

## Objetivo

Enseñar los fundamentos de HTML y CSS mediante un curso corto, visual y práctico. El curso está dirigido a estudiantes de 12 a 17 años sin experiencia previa en programación.

## Público objetivo

Estudiantes de secundaria sin experiencia previa en desarrollo web.

## Duración

Aproximadamente 60 minutos, distribuidos en 8 lecciones cortas.

## Prerrequisitos

Ninguno. El curso está diseñado para principiantes absolutos.

## Herramientas

El curso se realizará usando herramientas gratuitas y en línea. No se requiere instalar programas.

Herramienta principal:

- CodePen: https://codepen.io/pen/

Herramientas alternativas:

- JSFiddle: https://jsfiddle.net/
- Replit: https://replit.com/

## Requisitos para estudiantes

Solo se necesita un navegador web moderno:

- Chrome
- Firefox
- Safari
- Edge

## Estructura del repositorio

- course-plan.md: plan completo del curso.
- starter-files/: archivos iniciales para que los estudiantes practiquen.
- completed-examples/: ejemplos completos de cada lección.
- scripts/: guiones de cada video.
- esources/: enlaces, imágenes y recursos auxiliares.
- ideos/: enlaces a videos no listados de YouTube.
- docs/: documentos internos de organización del equipo.
- exports/: archivos generados para la entrega final.

## Lecciones

1. ¿Qué es una página web?
2. Estructura básica de HTML.
3. Títulos y párrafos.
4. Listas, imágenes y enlaces.
5. Introducción a CSS.
6. Estilos simples con CSS.
7. Proyecto final: página de perfil personal.
8. Recomendaciones y errores comunes.
