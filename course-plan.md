# Fundamentos de Desarrollo Web

## Resumen del curso

Este curso introduce a estudiantes de secundaria a la creación de páginas web sencillas usando HTML y CSS. El curso no requiere instalaciones, porque todas las prácticas se realizan desde un navegador web usando CodePen.

**Duración total:** aproximadamente 60 minutos  
**Público objetivo:** estudiantes de 12 a 17 años sin experiencia previa en programación  
**Prerrequisitos:** ninguno  
**Herramientas necesarias:** navegador web moderno  
**Editor online principal:** CodePen  
**Startup:** lowcortisol  
**Producto:** cleanflow  
**Team Leader:** James  
**Repositorio de código fuente:** pendiente  

## Objetivo general

Lograr que el estudiante comprenda cómo se construye una página web básica usando HTML para el contenido y CSS para el diseño visual.

## Objetivos específicos

Al finalizar el curso, el estudiante será capaz de:

- Explicar qué es una página web.
- Diferenciar HTML y CSS.
- Crear una estructura básica de HTML.
- Usar etiquetas como h1, h2, p, ul, li, img y .
- Reconocer atributos como href, src, lt y class.
- Aplicar estilos simples con CSS.
- Usar selectores, propiedades y valores.
- Crear una tarjeta visual con CSS.
- Construir una página de perfil personal.
- Identificar errores comunes de HTML y CSS.

## Herramientas del curso

El curso usará herramientas gratuitas y en línea:

- CodePen: https://codepen.io/pen/
- JSFiddle: https://jsfiddle.net/
- Replit: https://replit.com/

Los estudiantes solo necesitan usar un navegador web moderno. No se requiere instalar editores de código, extensiones, servidores ni herramientas adicionales.

Finalmente, se construye una página de perfil personal y se explican errores comunes para principiantes.
