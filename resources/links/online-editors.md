# Editores online

Este curso está diseñado para desarrollarse sin instalar herramientas. Todas las prácticas pueden realizarse desde un navegador web.

## Editor principal

### CodePen

https://codepen.io/pen/

CodePen será el editor online principal usado durante las grabaciones del curso. Permite escribir HTML, CSS y visualizar el resultado en tiempo real.

## Editores alternativos permitidos

### JSFiddle

https://jsfiddle.net/

### Replit

https://replit.com/

## Nota para estudiantes

Puedes usar CodePen o cualquier editor online de desarrollo web de tu preferencia. Lo importante es que puedas escribir HTML y CSS sin instalar herramientas en tu computadora.
