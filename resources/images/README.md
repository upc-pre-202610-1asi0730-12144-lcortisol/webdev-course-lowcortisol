# Recursos visuales del curso

Esta carpeta centraliza las imágenes usadas como apoyo documental para el curso de HTML y CSS.

## Logos disponibles

| Archivo | Representa | Uso recomendado |
|---|---|---|
| `html-logo.png` | HTML | Estructura, contenido y etiquetas de una página web. |
| `css-logo.png` | CSS | Estilos, colores, espaciado y presentación visual. |

Los logos deben reutilizarse desde esta carpeta para evitar duplicados. No es necesario insertarlos todavía en los archivos HTML o CSS de las lecciones si esos archivos siguen pendientes de desarrollo por los integrantes.
