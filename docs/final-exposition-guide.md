# Guía para exposición final

El curso se titula: Crea tu primera página web con HTML y CSS.

Está dirigido a estudiantes de secundaria de 12 a 17 años sin experiencia previa en programación.

El curso tiene una duración aproximada de una hora y está dividido en ocho lecciones cortas.

Se utilizó CodePen como editor online porque no requiere instalación, es gratuito y permite visualizar el código HTML, CSS y el resultado en tiempo real.

La secuencia del curso fue progresiva. Primero se explicó qué es una página web y la estructura básica de HTML. Luego se enseñaron elementos comunes como títulos, párrafos, listas, imágenes y enlaces. Después se introdujo CSS para aplicar colores, fuentes, bordes y estilos simples. Finalmente, se construyó una página de perfil personal como proyecto final.

El trabajo fue distribuido entre cinco integrantes:

- James desarrolló las lecciones 1 y 2 y lideró la coordinación del equipo.
- Kirk desarrolló las lecciones 3 y 4.
- Bruno desarrolló las lecciones 5 y 6.
- Miguel desarrolló la lección 7.
- Jean desarrolló la lección 8 y apoyó en el cierre general del curso.

Todos los videos fueron pensados para un público principiante, usando lenguaje sencillo, ejemplos visuales y pausas de práctica.

Además, el código fuente se organizó en un repositorio público de GitHub con carpetas para archivos iniciales, ejemplos completados y guiones.

Como resultado, el estudiante puede crear una página web básica de perfil personal usando HTML y CSS sin instalar ninguna herramienta.
